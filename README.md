# 📚 Book_Alchemy

![Python](https://img.shields.io/badge/Python-3.x-blue.svg) ![Flask](https://img.shields.io/badge/Flask-3.1.3-lightgrey.svg) ![SQLite](https://img.shields.io/badge/SQLite-Database-blue.svg)

Book_Alchemy is a lightweight web-based library management system built with Python, Flask, and SQLAlchemy. It provides a clean, responsive interface to track authors and their books, complete with searching, sorting, and management capabilities.

## ✨ Features
*   **Author Management**: Easily add new authors with their birth and death dates.
*   **Book Cataloging**: Manage book inventory, linking them directly to specific authors.
*   **Smart Search**: Filter your collection by title or author name.
*   **Sorting Capabilities**: View your library sorted by Title, Author, or Publication Year.
*   **Responsive UI**: Clean interface using custom CSS and Flask templates.
*   **Cover Integration**: Automatically fetches book covers via Open Library/Google Books APIs.

---

## 🛠 Tech Stack
*   **Backend**: Python, Flask, Flask-SQLAlchemy
*   **Database**: SQLite
*   **Frontend**: HTML5, CSS3, Jinja2 Templates

## 🚀 Installation
1. **Clone the repository**:
   ```bash
   git clone https://github.com/FrankStremmler/Book_Alchemy
   cd Book_Alchemy
   ```
2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```
3. **Run the application**:
   ```bash
   python app.py
   ```
   The application will be accessible at `http://127.0.0.1:5000`.

## 📖 Usage
*   **Home Page**: Browse your book collection. Use the search bar to find specific entries and the sorting buttons to organize the list.
*   **Add Author**: Navigate to `/add_author` to register a new author.
*   **Add Book**: Navigate to `/add_book` to add a book. Ensure an author is already created, as books require an author mapping.
*   **Delete**: Use the trash icon 🗑️ on any book card to remove it from the database.
